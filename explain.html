KL Divergence ≥ 0 (Gibbs' Inequality)

What's proved: The "distance" between any two probability distributions is always non-negative. You can never have a negative information gap. This means: no model can be "better than perfect" — the true distribution is always the best predictor of itself.

Why it matters: This is the foundation of all LLM training. The loss function (cross-entropy) always has a floor — the entropy of the true data. Training pushes toward this floor but can never go below it.
Lemma kl_nonneg : ∀ p q, is_distribution p → is_distribution q → kl_divergence p q ≥ 0.
Proof.
  unfold kl_divergence, entropy.
  pose proof (gibbs_inequality p q). (* H(p,q) ≥ H(p,p) *)
  lra. (* KL = H(p,q) - H(p) ≥ 0 ✓ *)
Qed.

Attention as a Simplicial Complex

What's proved: When tokens attend to each other above a threshold, they form geometric shapes (simplices). Any subset of a mutually-attending group also mutually attends — this is the "face preservation" property. The collection of all such groups forms a valid simplicial complex.

Why it matters: This gives attention a rigorous geometric/topological structure. You can compute "holes" (Betti numbers) in the attention pattern — connected components = separate topics, loops = circular reasoning, voids = knowledge gaps.

Lemma is_simplex_face_preservation :
  ∀ attn simplex face threshold,
    is_simplex attn simplex threshold →
    incl face simplex →
    is_simplex attn face threshold.
Proof.
  intros. unfold is_simplex in *.
  intros i j Hi Hj Hneq.
  (* face ⊆ simplex, so i,j ∈ simplex *)
  apply H_incl in Hi, Hj.
  apply H_sim; assumption. (* ✓ *)
Qed.

Feature Independence & Intervention Algebra

What's proved: If concept directions are orthogonal, then "ablating" (removing) one feature doesn't affect any other feature. Steering one feature to a target value works perfectly without side effects. This is the math behind activation patching and steering vectors.

Why it matters: This is the formal foundation of mechanistic interpretability (SAEs, probing, representation engineering). Orthogonal features = independent knobs you can turn. The proof shows: if features are orthogonal, interpretability is exact.
90°
Theorem ablation_preserves_orthogonal :
  ∀ model dict ctx i j ci cj,
    vec_dot(ci, cj) = 0 → (* orthogonal! *)
    concept_activation cj (ablate_feature ... i) =
    concept_activation cj (transformer_forward model ctx).
Proof.
  rewrite vec_dot_add_r, vec_dot_scale_r.
  rewrite (vec_dot_comm _ cj ci).
  rewrite Horth. (* dot = 0 → leakage = 0 *)
  lra. (* ✓ *)
Qed.

Transformer Layers Preserve Sequence Length

What's proved: Every component of a transformer (attention, FFN, residual connections, layer norm) preserves the number of tokens. What goes in with N tokens comes out with N tokens. This is proved by structural induction through all layers.

Why it matters: This guarantees the transformer's architecture is well-formed — you can stack layers without worrying about dimension mismatches. It's a type-safety proof for neural network architecture.
Theorem apply_all_layers_preserves_length : ∀ layers ctx,
  length (apply_all_layers layers ctx) = length ctx.
Proof.
  induction layers as [| l rest IH].
  - simpl. reflexivity. (* base: no layers = no change *)
  - simpl. rewrite IH.
    apply transformer_layer_preserves_length. (* inductive step ✓ *)
Qed.

Model Continuity & Knowledge Topology

What's proved: Small changes to the input context produce small changes in the output distribution (continuity). Open sets in context space are upward-closed under the prefix ordering (Alexandrov topology). Intersections and unions of open sets remain open.

Why it matters: Continuity means the model won't hallucinate wildly from a tiny input change. The topology gives "knowledge regions" — neighborhoods where the model behaves consistently. This is why similar prompts give similar answers.

Lemma intersection_upward_closed : ∀ U V,
  upward_closed U → upward_closed V → upward_closed (U ∩ V).
Proof.
  intros U V HU HV c1 c2 Hin Hpre.
  inversion Hin. constructor.
  - apply (HU c1); assumption.
  - apply (HV c1); assumption. (* ✓ *)
Qed.

Self-Modeling & Fixed Points

What's proved (partially): A sufficiently large model can approximately predict its own output — it can "simulate itself." This is a fixed-point theorem. But PERFECT self-prediction is impossible (diagonal argument, like Gödel's incompleteness).

Why it matters: This explains meta-cognition in LLMs — they can reason about what they'd say. But there's always a "self-knowledge gap." No model can perfectly know itself on all inputs, just like no formal system can prove its own consistency.
